\# 🌌 COSMOS — Образовательная платформа о космосе



Веб-приложение для изучения Солнечной системы, галактик, чёрных дыр и спутников. Включает 3D-модели планет, квиз, мини-игры и интерактивные карты.

\## 📖 Описание



\*\*COSMOS\*\* — интерактивный образовательный сайт о космосе с:

8 планетами Солнечной системы

Чёрными дырами + симулятор аккреции

8 спутниками (МКС, Хаббл, JWST, Voyager)

14 галактиками с поиском

3D-моделями на Three.js

Интерактивной картой (3 режима)

Калькулятором веса

Квизом из 12 вопросов

7 мини-играми

\-Админ-панелью



\*\*Стек:\*\* Python + Flask + SQLite (бэк) + Vanilla JS + Three.js (фронт)

\## 🚀 Быстрый старт (Windows)



Самый простой способ — \*\*просто запустить `start.bat`\*\*:



1\. Дважды кликните на `start.bat`

2\. Скрипт автоматически:

&#x20;  - Создаст виртуальное окружение

&#x20;  - Установит зависимости

&#x20;  - Заполнит БД

&#x20;  - Запустит сервер

3\. Откройте браузер: \*\*http://localhost:5000\*\*
